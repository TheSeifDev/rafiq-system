# Admin Dashboard Guide

## Access
URL: `/admin`

## Features

### Blog Manager (`/admin/blog`)
- Create/edit/delete posts
- Sync with Facebook page
- View engagement stats (likes, comments, shares)

### Reviews Manager (`/admin/reviews`)
- View all reviews
- Approve/reject reviews
- Feature spotlight reviews

### Team Manager (`/admin/team`)
- Add/edit team members
- Categorize by skill type (soft/hard)
- Toggle active status

### Links Manager (`/admin/links`)
- Manage social/contact links
- Custom SVG icons
- Navigation visibility toggle

### Services Manager (`/admin/services`)
- CRUD services
- Icon selection
- Display ordering

### Experience Manager (`/admin/experience`)
- Add work experience
- Timeline management

### Media Manager (`/admin/media`)
- Upload/manage images
- Gallery organization

## Authentication
- Login: `/login`
- Register: `/register`
- Logout: Available in admin sidebar
