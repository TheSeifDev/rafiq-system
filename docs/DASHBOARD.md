# Dashboard System

## Overview
Admin dashboard for managing all website content with dark theme UI.

## Sidebar Navigation
| Menu Item | Route | Description |
|-----------|-------|-------------|
| Dashboard | `/admin` | Main dashboard with stats |
| Reviews | `/admin/reviews` | Manage customer reviews |
| Blog | `/admin/blog` | Manage blog posts + Facebook sync |
| Experience | `/admin/experience` | Manage work experience |
| Media | `/admin/media` | Upload and manage files |
| Notifications | `/admin/notifications` | View system notifications |
| Settings | `/admin/settings` | Admin settings |

## User Profile
- Admin User: 
- Role: ADMIN
- Status: Online

## Media Library Features
- Upload files
- Preview images
- File details: Name, Bucket, Size, Used In, Uploaded date
- Pagination
- Search files

## Notifications
- Badge shows unread count (3 in screenshot)
- Real-time notifications
