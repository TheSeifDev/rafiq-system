# ADR-003: SSE Communication Bridge

## Context
The React/Electron GUI needs real-time streaming updates from the Python backend (e.g. speech transcription states, voice levels, dynamic reminders).

## Decision
Use HTTP Server-Sent Events (SSE) from FastAPI to push real-time event packets to the frontend UI.

## Consequences
* Unidirectional low-overhead stream.
* Simpler than WebSockets since frontend only needs to read status updates.
