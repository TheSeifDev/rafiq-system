# ⚙️ Feature Flags (Configuration Settings)

Controlled via local `.env`:
* `RAFIQ_AUTO_RECOVERY`: Enable self-restart when API timeout counts exceed threshold.
* `RAFIQ_BARGE_IN`: Allow voice input to interrupt text-to-speech outputs.
* `RAFIQ_ENABLE_WEB_SEARCH`: Toggle WHO RAG search capability.
* `RAFIQ_DISABLE_PRIVACY`: Turn off `Pseudonymizer` filter for local debug runs.
