# 🗺️ Roadmap: Rafiq System

* **Phase 1: MVP Stability (Current)**: Refine audio capture stability, eliminate PyAudio thread locks, and implement auto-recovery hooks.
* **Phase 2: Local AI Inference**: Add support for local Whisper and Llama model execution to enable full offline capability.
