# 🆔 Project Identity: رفيق (Rafiq System)

**رفيق** (Rafiq - meaning companion in Arabic) is a specialized AI assistant designed to provide medical and psychological support for home healthcare.

## Brand Values
1. **Empathy**: Conversing in natural, reassuring tones (Arabic dialects and English).
2. **Clinical Precision**: Strict reliance on WHO evidence and RxNav safety databases.
3. **Privacy First**: Ensuring no patient personal data leaves the local machine.
4. **Reliability**: A system that heals itself, recovers from API failures, and logs warnings immediately.

## Target Demographics
* **Elderly Patients**: Easy-to-use voice interface requiring zero text typing.
* **Chronic Patients**: Continuous medication tracking, reminder snoozing, and low-stock alerts.
* **Caregivers**: Direct status tracking and safety event logging.
