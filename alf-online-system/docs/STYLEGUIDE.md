# 🎨 Styleguide

## Python Style
* Format code with Black.
* Use explicit type annotations for all core module signatures.
* Always clean up open file descriptors using context managers (`with` statements).

## Frontend Style
* CSS variables must be defined in `index.css`.
* Use Shadcn UI primitives for inputs, dialogs, and progress bars.
* Components must use React 19 rules (e.g. clean hook dependency arrays).
