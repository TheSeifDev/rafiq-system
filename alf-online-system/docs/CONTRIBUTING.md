# 🤝 Contributing to Rafiq System

## Code Standards
* **Python**: Enforce PEP 8. Use type hints for all public functions. Return async coroutines for I/O operations.
* **TypeScript/React**: Enforce component modularity. Use Tailwind for style system variables.
* **Commit Messages**: Prefix commits with `feat:`, `fix:`, `docs:`, or `refactor:`.

## Branching Workflow
1. Create a feature branch off `main`.
2. Implement code and add unit tests.
3. Run `pytest` to verify all test suites pass.
4. Open a Pull Request for architectural review.
